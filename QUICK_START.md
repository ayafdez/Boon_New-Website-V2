## Get Running in 3 Minutes

### Step 1: Install Dependencies (1 minute)

```bash
cd boon-website-complete
npm install
```

### Step 2: Start Development Server (30 seconds)

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

**You should see the Homepage!**

### Step 3: Test All Pages

Navigate to:
- Homepage: `http://localhost:5173/`
- Solutions: `http://localhost:5173/solutions`
- For Coaches: `http://localhost:5173/for-coaches`
- For Individuals: `http://localhost:5173/for-individuals`
- Resources: `http://localhost:5173/resources`

## Deploy to GitHub + Vercel (5 minutes)

### 1. Push to GitHub

```bash
# Initialize git
git init

# Add all files
git add .

# Commit
git commit -m "Boon website - complete"

# Create main branch
git branch -M main

# Add your remote (create repo on GitHub first!)
git remote add origin https://github.com/boon-coaching/YOUR-REPO-NAME.git

# Push
git push -u origin main
```

### 2. Deploy on Vercel

1. Go to [https://vercel.com](https://vercel.com)
2. Click "Add New Project"
3. Import your GitHub repository
4. Click "Deploy" (Vercel auto-detects Vite)
5. **Done!** Your site is live in ~2 minutes
